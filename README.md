

# WebSocket,Next,React Konva를 사용한 간단한 그림판 프로젝트

## 프로젝트 설명
WebSocket 사용해보는 목적 겸 쇼핑물 같은 사이트 말고 게임이나 그림판 같은 평소에 하지 않았던 내용을 배우고 싶어서 시작한 프로젝트입니다. 

간단하게 그리는 기능과 지우기 기능 ctrl z  ctrl y 와 같은 기능이 존재합니다.
